# Task 001: Project scaffold - pyproject.toml, package structure, ruff, mypy, pytest

## Objective

Create the Python package skeleton for the dkg project. This task produces no application logic - only the project configuration, directory structure, a smoke test, and a locked dependency set that all subsequent tasks build on.

## Acceptance Criteria

- [ ] `uv sync --frozen` succeeds on a clean checkout with no manual steps
- [ ] `uv run ruff check .` exits 0
- [ ] `uv run ruff format --check .` exits 0
- [ ] `uv run mypy src/` exits 0 with no issues
- [ ] `uv run pytest tests/test_smoke.py -v` exits 0 and reports 1 passed
- [ ] `src/dkg/__init__.py` exports `__version__ = "0.1.0"`

## Outputs

- pyproject.toml
- ruff.toml
- mypy.ini
- uv.lock
- src/dkg/__init__.py
- tests/test_smoke.py

## Requirements

- Package name: dkg, version 0.1.0
- Python requires: >= 3.11
- Dev dependencies: ruff, mypy, pytest
- Runtime dependencies: numpy, scipy, polars, pyarrow, joblib, networkx, pydantic
- ruff: target-version = "py311", line-length = 100, select = ["E", "F", "I", "UP"]
- mypy: strict = true
- Run `uv lock` and commit the resulting uv.lock - do not just create the file manually
- Smoke test imports `dkg` and asserts `dkg.__version__ == "0.1.0"`
