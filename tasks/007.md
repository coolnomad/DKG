## Requirements

- Fit linear model via scipy.stats.linregress
- Fit natural cubic spline of df=spline_df=3 via patsy ns() or scipy; use OLS on spline basis
- F-test for nonlinearity: F = ((RSS_linear - RSS_spline) / (df_spline - df_linear)) / (RSS_spline / (n - df_spline - 1)); p-value from scipy.stats.f.sf
- monotonicity_score = abs(mean(sign(diff(spline_pred sorted by x))))
- mean_shape_direction = sign(median(diff(spline_pred sorted by x)))
- spline_pred_q10_to_q90: spline predicted value at x=Q90 minus x=Q10
- spline_pred_range: max - min of all spline fitted values
- spline_direction_changes: number of sign changes in consecutive spline diffs
- delta_r2 = spline_r2 - linear_r2; delta_aic = AIC(linear) - AIC(spline)
- All field names match R
