## Requirements

- n_folds=5 cross-validation; fold assignment via numpy.random.default_rng(config.seed)
- Fit linear OLS and natural cubic spline (spline_df=3) per fold; record out-of-fold predictions
- Continuous metrics: null_rmse=sd(target), linear_rmse, spline_rmse, delta_rmse, linear_mae, spline_mae, linear_cv_r2, spline_cv_r2, linear_skill_score=(null_rmse-linear_rmse)/null_rmse
- Left-tail event at left_tail_threshold (default Q10 of target); fit logistic regression per fold via sklearn LogisticRegression or statsmodels; compute AUROC, PR-AUC, Brier score, pr_auc_lift = pr_auc / (prevalence + eps)
- Second logistic model using Q20 threshold; report left_tail_auc_q20, left_tail_pr_auc_q20, left_tail_brier_q20, pr_auc_lift_q20
- AUROC via trapezoidal rule on sorted (fpr, tpr); PR-AUC via trapezoidal rule on (recall, precision)
- Return insufficient_data sentinel row if n < 30, unique predictor values < 10, or sd(target) == 0
- All field names match R
