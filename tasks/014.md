# Task 014: Phase 10 port — bootstrap stability

## Objective

Port summarize_relationship_stability() from fitting_functions.R to dkg.phases.phase10. Output is long-format with one row per metric per resampling source.

## Acceptance Criteria

- [ ] Output DataFrame has columns: predictor, target, n, source, metric, mean, sd, q025, median, q975, ci_width, relative_cv, sign_consistency, sign_positive_frac, sign_negative_frac, n_success
- [ ] Both bootstrap and subsample sources present for all metrics
- [ ] All 12 tracked metrics present: linear_slope, linear_r2, robust_slope, left_tail_risk_ratio, left_tail_risk_difference, regime_threshold, regime_delta_aic, regime_delta_r2, wasserstein_1, ks_statistic, mean_shift, median_shift
- [ ] ci_width, relative_cv, sign_consistency derived columns correct
- [ ] All default checks pass

## Outputs

- src/dkg/phases/phase10.py
- tests/test_phase10.py

## Requirements

- Two resampling schemes: bootstrap (n_boot samples with replacement, size=n) and subsampling (n_boot samples without replacement, size=floor(n*sample_frac))
- Per resample run four full phase functions: summarize_phase2, summarize_phase5, summarize_phase7, summarize_phase8; extract the 12 tracked metrics from combined outputs
- Catch exceptions per resample and record None; n_success = count of successful resamples for that metric
- Summarize each metric: mean, sd, q025 (2.5th pct), median (50th pct), q975 (97.5th pct), sign_positive_frac, sign_negative_frac
- ci_width = q975 - q025
- relative_cv = sd / (abs(mean) + 1e-8)
- sign_consistency = max(sign_positive_frac, sign_negative_frac)
- Seed all randomness via numpy.random.default_rng(config.seed)
- Output has 24 rows per pair (12 metrics x 2 sources)
- Return insufficient_data sentinel if n < 30, unique predictor values < 10, or sd(target) == 0
- All field names match R
