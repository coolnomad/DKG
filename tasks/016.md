# Task 016: Tier 3 pipeline orchestrator

## Objective

Implement dkg.tier3.run_stability() which selects top-K pairs from Tier 2, runs Phase 10 bootstrap stability on each, and writes tier3_stability.parquet.

## Acceptance Criteria

- [ ] `run_stability(tier2_df, X, Y, x_cols, y_cols, config)` returns a Polars DataFrame
- [ ] Top-K pairs selected by abs(pearson_r) descending from tier2_df
- [ ] Output is long-format Phase 10 results with x_col and y_col prepended
- [ ] tier3_stability.parquet written to config.output_dir
- [ ] joblib parallelism used
- [ ] All default checks pass

## Inputs

- src/dkg/tier2.py
- src/dkg/phases/phase10.py

## Outputs

- src/dkg/tier3.py
- tests/test_tier3.py

## Requirements

- Select top config.top_k pairs from tier2_df ranked by abs(pearson_r) descending; `pearson_r` is carried forward from Tier 1 into tier2_df by run_deep() — no separate tier1_df reference needed
- For each selected pair run summarize_phase10; collect results into list of DataFrames
- Concatenate all Phase 10 long-format results; prepend x_col and y_col columns
- Write to tier3_stability.parquet
- Use joblib.Parallel(n_jobs=config.n_jobs, backend="loky")
