# Task 011: Phase 7 port — regime and threshold structure

## Objective

Port summarize_regime_threshold_structure() from fitting_functions.R to dkg.phases.phase7.

## Acceptance Criteria

- [ ] Output dict contains all fields produced by the R function with matching names
- [ ] Best threshold selection uses delta_aic (not RSS minimization)
- [ ] threshold_stability, threshold_quantile, slope_sign_change all present
- [ ] left_tail_fisher_p and regime_median_shift present
- [ ] All default checks pass

## See Also

- fitting_functions.R

## Outputs

- tests/test_phase7.py

## Requirements

- Scan n_thresholds=25 candidate thresholds evenly spaced across predictor quantiles [min_quantile=0.10, max_quantile=0.90]
- For each threshold t, define regime = predictor > t; fit piecewise OLS: target ~ predictor + regime + predictor:regime via numpy or statsmodels; record piecewise_r2, piecewise_aic, delta_r2, delta_aic = linear_aic - piecewise_aic
- Select best threshold by maximum delta_aic (NOT minimum RSS)
- At best threshold record: pre/post slopes, slope_difference, slope_sign_change = sign(pre) != sign(post), left_tail_risk_ratio and risk_difference, left_tail Fisher exact p-value via scipy.stats.fisher_exact, low/high regime variance, variance_ratio, sd_ratio_regimes = sqrt(variance_ratio), regime_median_shift
- threshold_stability = fraction of scan grid where delta_aic > 0
- threshold_quantile = mean(predictor <= best_threshold)
- Require min 10 observations in each regime; skip thresholds that do not meet this
- Return insufficient_data sentinel row if n < 20, unique predictor values < 10, or sd(target) == 0
- All field names match R
