# Task 010: Phase 6 port — skewness and asymmetry structure

## Objective

Port summarize_skewness_asymmetry_structure() from fitting_functions.R to dkg.phases.phase6.

## Acceptance Criteria

- [ ] Output dict contains all fields produced by the R function with matching names
- [ ] global_skew and skew_slope match R within atol=1e-6 on a synthetic test pair
- [ ] skew_sign_change boolean present
- [ ] asymmetry_slope and global_asymmetry_index present
- [ ] All default checks pass

## See Also

- fitting_functions.R

## Outputs

- src/dkg/phases/phase6.py
- tests/test_phase6.py

## Requirements

- Bin predictor into n_bins=4 quantile bins
- Compute per-bin skewness of the RAW target values (not residuals) in each bin: mean((x-mean)^3)/sd^3
- Compute skew_slope as slope of linear fit of bin_skew ~ bin_median_x via scipy.stats.linregress
- skew_direction = sign(skew_slope)
- skew_sign_change = True if sign(low_x_skew) != sign(high_x_skew) where low/high split at Q25/Q75 of predictor
- Quantile asymmetry index per bin: (lower_spread - upper_spread) / (lower_spread + upper_spread + eps) where lower_spread = Q50 - lower_q (default 0.10), upper_spread = upper_q (default 0.90) - Q50
- asymmetry_slope = slope of linear fit of quantile_asymmetry_index ~ bin_median_x
- global_skew = skewness of full target vector
- global_asymmetry_index = (global_lower_spread - global_upper_spread) / (global_lower_spread + global_upper_spread + eps)
- Return insufficient_data sentinel row if n < 10, unique predictor values < 4, or sd(target) == 0
- All field names match R
