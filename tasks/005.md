# Task 005: Phase 1 port — vector geometry summary

## Objective

Port summarize_vector_geometry() from fitting_functions.R to dkg.phases.phase1. Add LOO robustness checks for the three moment-based shape statistics not present in the R version.

## Acceptance Criteria

- [ ] Output dict contains all fields produced by the R function with matching names
- [ ] LOO fields present: skewness_loo_max_influence, kurtosis_loo_max_influence, bimodality_loo_max_influence, skewness_is_robust, kurtosis_is_robust, bimodality_is_robust
- [ ] `sweep_phase1(matrix, col_names, config)` returns a Polars DataFrame with one row per column
- [ ] `filter_by_geometry(df, config)` returns the DataFrame with geometry_keep and geometry_filter_reason columns added
- [ ] Constant or all-NaN vectors return a sentinel row with geometry_status != "ok" rather than raising
- [ ] All default checks pass

## Outputs

- src/dkg/phases/phase1.py
- tests/test_phase1.py

## Requirements

- Port all fields from summarize_vector_geometry(): completeness counts, quantiles (0/1/5/10/25/50/75/90/95/99/100), mean, sd, mad, IQR, zero_frac, near_zero_var, bin_n_min/max/imbalance (n_bins=5), skewness, excess kurtosis, bimodality coefficient (skew²+1)/kurtosis, Hartigan dip statistic and p-value (use diptest package or implement directly), density peak/valley count and ruggedness from scipy KDE, effective support size and fraction, left/right tail span, tail asymmetry ratio
- LOO robustness: for each of skewness, excess kurtosis, and bimodality coefficient, draw min(n, config.n_loo) subsamples of size n-1 without replacement using numpy.random.default_rng(config.seed); compute max absolute change across subsamples; set {stat}_is_robust = True if max_influence < config.loo_influence_threshold * abs(full_stat)
- `sweep_phase1(matrix: np.ndarray, col_names: list[str], config: RunConfig) -> pl.DataFrame` applies summarize_phase1 to each column
- `filter_by_geometry` mirrors R's filter_features_by_geometry() with same default thresholds
- Add diptest to project dependencies via uv add
