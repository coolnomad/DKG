# Task 012: Phase 8 port — distributional shift

## Objective

Port summarize_distributional_shift() from fitting_functions.R to dkg.phases.phase8.

## Acceptance Criteria

- [ ] Output dict contains all fields produced by the R function with matching names
- [ ] wasserstein_1 and ks_statistic match R within atol=1e-6 on a synthetic test pair
- [ ] Quantile shift profile columns present: q05_shift, q10_shift, q25_shift, q50_shift, q75_shift, q90_shift, q95_shift
- [ ] energy_distance present
- [ ] All default checks pass

## Outputs

- src/dkg/phases/phase8.py
- tests/test_phase8.py

## Requirements

- Split target into high-X (>= x_quantile_cut=0.75 quantile) and low-X (<= 1-x_quantile_cut quantile) groups
- KS test via scipy.stats.ks_2samp; record ks_statistic and ks_p
- Wasserstein-1: mean(abs(quantile_high(grid) - quantile_low(grid))) over 99-point grid from 0.01 to 0.99; signed_wasserstein_shift = mean(q_high - q_low)
- Energy distance: 2*E|X-Y| - E|X-X'| - E|Y-Y'| via pairwise mean absolute differences (outer product approach)
- Quantile shift profile at probs=[0.05, 0.10, 0.25, 0.50, 0.75, 0.90, 0.95]: q{pct}_shift = quantile_high[p] - quantile_low[p]
- quantile_profile_distance = sqrt(mean(q_diff²)); max_abs_quantile_shift
- mean_shift = mean(high_y) - mean(low_y); median_shift = median(high_y) - median(low_y)
- sd_ratio = sd(high_y) / sd(low_y); iqr_ratio = IQR(high_y) / IQR(low_y)
- shift_direction = sign(median_shift)
- tail_divergence_ratio = abs(q05_shift + eps) / abs(q95_shift + eps)
- quantile_shift_monotone_frac: fraction of consecutive q_diff steps in dominant direction
- Return insufficient_data sentinel row if n < 20, unique predictor values < 4, or sd(target) == 0
- All field names match R
