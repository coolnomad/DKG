# Task 022: Tier 1 performance benchmark and optimization

## Objective

Profile Tier 1 on synthetic full-scale matrices and optimize until the 30-minute wall-time target is met, documenting the final configuration and achieved throughput.

## Acceptance Criteria

- [ ] benchmarks/bench_tier1.py runs without error on synthetic 220x20000 X and 220x12000 Y matrices
- [ ] Wall time, peak RAM, and throughput (pairs/second) are measured and printed
- [ ] benchmarks/README.md documents final achieved throughput, hardware spec, and all optimizations applied
- [ ] Optimized path matches non-optimized within atol=1e-7 on a 100-column test case
- [ ] All default checks pass

## Inputs

- src/dkg/tier1.py

## Outputs

- benchmarks/bench_tier1.py
- benchmarks/README.md

## Requirements

- Synthetic matrices: numpy.random.default_rng(42).normal(size=(220, 20000)) for X, size=(220, 12000) for Y
- Measure wall time via time.perf_counter; peak RAM via tracemalloc or psutil
- If 30-minute target not met, apply optimizations in priority order: (1) float32 downcast before matrix multiply, (2) column-block chunking to reduce peak memory, (3) BLAS thread tuning via threadpoolctl
- Document memory estimate in bench script: 220x20K float32 ~17MB per matrix; 20Kx12K output correlation matrix ~960MB float32; chunking may be mandatory
- benchmarks/README.md records: hardware description, achieved wall time, peak RAM, pairs/second, list of optimizations applied
