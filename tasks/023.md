# Task 023: End-to-end integration test

## Objective

Add tests/test_integration.py with a smoke test that runs unconditionally and a full-scale integration test gated on real DepMap data, validating the complete pipeline end-to-end.

## Acceptance Criteria

- [ ] Smoke test runs unconditionally in CI on synthetic 220x500 X and 220x300 Y matrices and completes in under 60 seconds
- [ ] Smoke test produces all expected output files: tier0_marginals.parquet, tier1_screen.parquet, tier2_deep.parquet, tier3_stability.parquet
- [ ] Smoke test verifies all output files are non-empty and schema matches expected column lists
- [ ] Full-scale test is skipped automatically if --depmap-path not provided
- [ ] Full-scale xx mode test asserts PPARG and RXRA appear in the same Louvain community in communities.parquet
- [ ] All default checks pass

## Inputs

- src/dkg/modes/xy.py
- src/dkg/modes/xx.py
- src/dkg/cli.py

## Outputs

- tests/test_integration.py

## Requirements

- Smoke test: generate synthetic 220x500 X and 220x300 Y matrices via numpy.random.default_rng(42); run xy mode via modes.xy.run(); assert all four Parquet files exist and are non-empty; assert tier1_screen.parquet has columns x_col, y_col, pearson_r, pearson_p, spearman_r, spearman_p, n_obs; must complete under 60 seconds
- Full xy test: gated on --depmap-path; runs xy mode on real matrices; asserts non-empty outputs and schema correctness; wall-time limit 35 minutes
- Full xx community test: gated on --depmap-path; runs xx mode on predictor matrix; loads communities.parquet; asserts PPARG and RXRA share the same community_id
- Use pytest tmp_path fixture for all output directories
- All gated tests use pytest.mark.skipif on --depmap-path fixture

## Output Validation

- uv run pytest tests/test_integration.py -v -k "smoke"
