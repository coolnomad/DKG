## Requirements

- Split into high-X (>= x_quantile_cut=0.75 quantile) and low-X (<= 1-x_quantile_cut quantile) groups
- Define left-tail event as target <= left_threshold (default Q10 of target); right-tail as target >= right_threshold (default Q90)
- Compute event rates in each X group for both tails: left_rate_low_x, left_rate_high_x, right_rate_low_x, right_rate_high_x
- Risk ratio = (high_rate + eps) / (low_rate + eps); risk difference = high_rate - low_rate
- dominant_tail_direction = sign(left_risk_diff - right_risk_diff) * -1
- Fisher exact test on 2x2 contingency table (x_group x tail_event) via scipy.stats.fisher_exact
- Bin predictor into n_bins=4; compute per-bin left/right event rates, min/max y, Q05/Q95 y; bin_left_rate_monotone_frac
- Record event counts: n_left_tail_low_x, n_left_tail_high_x, n_right_tail_low_x, n_right_tail_high_x
- Return insufficient_data sentinel row if n < 10, unique predictor values < 4, or sd(target) == 0
- All field names match R
