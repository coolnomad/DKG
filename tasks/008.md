# Task 008: Phase 4 port — conditional variance structure

## Objective

Port summarize_conditional_variance_structure() from fitting_functions.R to dkg.phases.phase4.

## Acceptance Criteria

- [ ] Output dict contains all fields produced by the R function with matching names
- [ ] abs_resid_slope and sq_resid_slope match R values within atol=1e-6 on a synthetic test pair
- [ ] variance_ratio_high_low and sd_ratio_high_low present and correct
- [ ] Both mean_model="linear" and mean_model="spline" variants work
- [ ] All default checks pass

## See Also

- fitting_functions.R

## Outputs

- src/dkg/phases/phase4.py
- tests/test_phase4.py

## Requirements

- Fit mean model (linear or spline per mean_model param); compute residuals
- Fit OLS of abs(resid) ~ predictor via scipy.stats.linregress; record abs_resid_slope, abs_resid_slope_p, abs_resid_r2
- Fit OLS of resid² ~ predictor; record sq_resid_slope, sq_resid_slope_p, sq_resid_r2
- Compute Spearman correlation of predictor vs abs(resid) and resid² via scipy.stats.spearmanr
- Split into low-X (bottom quartile) and high-X (top quartile) groups; compute variance and IQR for each; record variance_ratio_high_low, sd_ratio_high_low = sqrt(variance_ratio), iqr_ratio_high_low, variance_direction = sign(high_var - low_var)
- Bin predictor into n_bins=4 quantile bins; compute per-bin variance and SD; record bin_var_ratio, bin_iqr_ratio, bin_sd_monotone_frac, bin_sd_range
- Return insufficient_data sentinel row if n < 10, unique predictor values < 4, or sd(target) == 0
- All field names match R
